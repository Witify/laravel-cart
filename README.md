## Installation

You can install the package via composer:

```bash
composer require witify/laravel-cart
```

## Usage

Add it to your controller:
``` php

use Witify\LaravelSeoAttributes\SeoTrait;

class HomeController extends Controller {
    use SeoTrait;
}
```

### Testing

``` bash
composer test
```
